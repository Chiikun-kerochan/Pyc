①注意事項

このプログラムで損害をうけるだなんてチャップリンですら無いとは思いますが、私は使用者がいかなる損害を受けようとも一切その責任は負いかねます。

②環境やモジュール
Python 3.11.0（3以上なら多分大丈夫）
Pytchat 0.5.5
Pandas
dotenv
shutil

③下準備
まずpythonの導入。
https://camp.trainocate.co.jp/magazine/python-install
上のやつで見てほしいな。

次に他のモジュールについて
コマンドプロンプトで下のやつを1行ずつ打ち込んでインストール
pip install pytchat
pip install pandas
pip install python-dotenv

④使い方
.envというファイルに、取り込みたい動画のURLを設定。そしたらPhalenkun-comment.pyを実行。
そうするとcsvファイルが同じディレクトリに作成される。以上。
csvファイルを作成したら、必ず名前を変えること。同じ名前のファイルがあると正常に動きません。

⑤その他
comment取得ファイルはデスクトップに置いといてください.
また、初期設定ではメインのｐｙファイルのディレクトリが私のｐｃのディレクトリになっているので任意のディレクトリに変えてください。
もしくはその行を消すことでcomment取得のところにcsvファイルが作成されます。
